# HK Recycling Sources Used

## Official sources

- **GREEN@COMMUNITY overview**
  - https://www.wastereduction.gov.hk/en-hk/waste-reduction-programme/green-at-community
  - 12 Recycling Stations
  - 82 Recycling Stores
  - about 600 Recycling Spots
  - supports 9 common recyclables

- **Clean Recycling**
  - https://www.wastereduction.gov.hk/en-hk/waste-reduction-programme/clean-recycling
  - Paper: keep dry, remove non-paper parts
  - Plastic bottles: remove caps/labels, rinse
  - Metal cans: empty, rinse
  - Beverage cartons: remove straw/wrapping, cut, rinse, dry, flatten

- **Home Recycling One Stop Shop**
  - https://www.wastereduction.gov.hk/en-hk/one-stop-shop
  - central page for finding where and when to recycle
  - points to map + GREEN@COMMUNITY opening hours

- **Food Waste Recycling Schemes**
  - https://www.wastereduction.gov.hk/en-hk/waste-reduction-programme/food-waste-recycling-schemes
  - food waste points at public refuse collection points
  - food waste spots for nearby residents/businesses

- **Rechargeable Battery Recycling Programme**
  - https://www.wastereduction.gov.hk/en-hk/waste-reduction-programme/rechargeable-battery-recycling-programme
  - batteries need designated collection, not normal public recycling bins

- **Map of Recycling Points / recycling-point pages**
  - https://www.wastereduction.gov.hk/en-hk/recycling-map
  - useful for glass collection points and district-specific facilities

- **Waste Less app notice**
  - https://www.wastereduction.gov.hk/en-hk/waste-less-mobile-app-notice-discontinuation
  - note: Waste Less app is discontinued; use GREEN$ app + website tools

## How this project uses the sources

- **Public bins**:
  - clean plastic bottles
  - metal cans
  - paper/cardboard
  - beverage cartons can be treated as a dedicated route in the demo when needed

- **Designated points**:
  - glass
  - food waste
  - batteries
  - beverage cartons if you want to emphasize the special collection route

- **General waste**:
  - cigarette butts
  - dirty pizza boxes
  - plastic film
  - mixed/uncertain items
